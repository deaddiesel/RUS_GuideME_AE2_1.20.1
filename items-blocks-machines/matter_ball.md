# Материальные шарики

<ItemImage id="matter_ball" scale="4" />

Шарик из обычной материи, используемый в качестве боеприпаса для <ItemLink id="matter_cannon" /> или для создания [шариков с краской](paintballs.md).

Создаётся из 256 предметов или вёдер в <ItemLink id="condenser" /> в режиме материального шарика.

# Высекатель

<BlockImage id="inscriber" scale="8" />

Высекатель используется для вырезания схем и [процессоров](processors.md) используя [печати](presses.md), а также для измельчения различных предметов в пыль.
Он может принимать как энергию мода «Прикладная энергетика 2» (AE) или энергию Fabric/Forge (E/FE). Он может быть двухсторонним, так что помещение предметов с разных сторон
будет помещать их в разные слоты инвентаря. Для этого его можно повернуть с помощью <ItemLink id="certus_quartz_wrench" />.
Его также можно настроить на перемещение результатов создания в соседние инвентари.

Размер входного буфера можно регулировать. Например, если вы хотите загрузить материалы из одного инвентаря в большой массив высекателей,
вам нужен небольшой буфер, чтобы материалы распределялись между высекателями более оптимально (вместо того, чтобы первый
высекатель заполнялся до 64, а остальные были пустыми).

Для создания [процессоров](processors.md) используются 4 печати для высекателя.

## Настройки

* Высекатель можно настроить на одностороннюю работу (как описано ниже) или разрешить вход в любой слот с любой стороны, при этом внутренний фильтр будет определять,
  что куда помещается. В одностороннем режиме предметы нельзя извлекать из верхнего и нижнего слотов.
* Высекатель можно настроить на перемещение предметов в соседние инвентари.
* Размер входного буфера можно регулировать: параметр «большой» предназначен для автономного высекателя с ручной загрузкой,
  параметр «малый» предназначен для повышения эффективности больших параллельных установок.

## GUI и многосторонность

В двухстороннем режиме высекатель фильтрует то, куда и с какой стороны вы помещаете или извлекаете.

![Inscriber GUI](../assets/diagrams/inscriber_gui.png) ![Inscriber Sides](../assets/diagrams/inscriber_sides.png)

A. **Верхний вход**, доступ через верхнюю часть высекателя (предметы можно как помещать в этот слот, так и извлекать из него).

B. **Центральный вход**, доступ через левую, правую, переднюю и заднюю части высекателя (предметы можно только помещать в этот слот, но не извлекать из него).

C. **Нижний вход**, доступ через нижнюю часть высекателя (предметы можно как помещать в этот слот, так и извлекать из него).

D. **Выход**, доступ через левую, правую, переднюю и заднюю части высекателя (предметы можно только извлекать из этого слота, но не помещать в него).

## Простая автоматизация

Например, возможность поворота и многосторонности позволяет полуавтоматизировать высекатели следующим образом:

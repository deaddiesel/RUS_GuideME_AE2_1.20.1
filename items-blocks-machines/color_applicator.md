# Аппликатор цвета

<ItemImage id="color_applicator" scale="4" />

Аппликатор цвета используется для покраски цветных блоков, таких как [кабели](cables.md), шерсть, терракота, стекло и бетон. Он использует
[шарики с краской](paintballs.md) или красители, а снежки можно использовать для удаления краски с кабелей и пятен от шариков с краской на блоках.

Его энергию можно зарядить в <ItemLink id="charger" />.

Аппликаторы цвета действуют как [ячейки хранения](storage_cells.md), и их запас краски проще всего пополнить, вставив
аппликатор в слот ячейки хранения в <ItemLink id="chest" />

Чтобы использовать аппликатор цвета, щёлкните [ПКМ] для применения и удерживайте [Shift + СКМ] для переключения между сохранёнными шариками и красителями.

## Улучшения

Аппликаторы цвета поддерживают следующие [улучшения](upgrade_cards.md), вставляемые через <ItemLink id="cell_workbench" />:

*   <ItemLink id="equal_distribution_card" /> выделяет одинаковое количество байтов для каждого типа, поэтому один тип не может заполнить всю ячейку.
*   <ItemLink id="void_card" /> аннулирует вставленные предметы, если ячейка заполнена (или выделенное пространство для конкретного типа в случае карты
    равномерного распределения). Будьте внимательны при разделении!
*   <ItemLink id="energy_card" /> для увеличения ёмкости аккумулятора.

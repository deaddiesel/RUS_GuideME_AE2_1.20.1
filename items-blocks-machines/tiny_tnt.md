# Крохотный динамит

<BlockImage id="tiny_tnt" scale="8" />

Небольшой динамит для небольших взрывов. Полезно для создания пар <ItemLink id="quantum_entangled_singularity" />.

Можно отключить ломание блоков в конфигурации, чтобы создавать сингулярности без возможности
гриферства, если вы хотите отключить динамит и криперов на своём сервере.

# Кристалл истинного кварца

<ItemImage id="certus_quartz_crystal" scale="4" />

*«Кристаллы истинного кварца обладают уникальной способностью поглощать большое количество энергии в своей кристаллической матрице».*

Один из основных ингредиентов для блоков мода «Прикладная энергетика 2», [устройств](../ae2-mechanics/devices.md), и предметов. Получается путём создания из [бутонов истинного кварца](../ae2-mechanics/certus-growth.md).

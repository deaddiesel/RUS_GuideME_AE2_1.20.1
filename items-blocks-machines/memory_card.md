# Карта памяти

<ItemImage id="memory_card" scale="4" />

Карта памяти используется для копирования и вставки настроек между [устройствами](../ae2-mechanics/devices.md) мода «Прикладная энергетика 2», и подключения
[P2P-тоннелей](p2p_tunnels.md). Также с её помощью можно вставлять [карты улучшения](upgrade_cards.md) в устройства.

- Удерживайте [Shift + ПКМ], чтобы скопировать настройки или создать новую частоту привязки P2P.
- Щёлкните [ПКМ], чтобы скопировать настройки или создать новую частоту привязки P2P.

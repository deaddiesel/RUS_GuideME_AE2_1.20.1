# Пространственный якорь

<BlockImage id="spatial_anchor" p:powered="true" scale="8"/>

Для функционирования любого из [устройств](../ae2-mechanics/devices.md) сети мода «Прикладная энергетика 2» должна быть загружена чанками. Если загружена только часть сети,
она может работать некорректно. Пространственный якорь решает эту проблему. Он принудительно загружает чанки, занимаемые сетью.
Одного кабеля, проходящего через границу чанка, достаточно для загрузки нового чанка.

Он распространит свою «загрузку» по [квантовым мостам](quantum_bridge.md), но не между измерениями, поэтому, если у вас
есть квантовый мост в Незер, вам нужен пространственный якорь как в сети в вашей базы, так и в сети в Незере.

По умолчанию он также включает случайные тики в загруженных чанках. Это можно отключить в конфигурации мода «Прикладная энергетика 2».

Его можно повернуть с помощью <ItemLink id="certus_quartz_wrench" />, если по какой-то причине вы хотите это сделать.

## Настройки

*	Пространственный якорь обеспечивает доступ к глобальным настройкам для просмотра энергии в AE или E/FE.
*	Возможность отображения внутренней голограммы, показывающей загружаемые блоки.

## Энергия

Пространственный якорь будет использовать [энергию](../ae2-mechanics/energy.md) согласно следующему уравнению:

e = 80 + (x\*(x+1))/2

где x — количество загружаемых чанков.

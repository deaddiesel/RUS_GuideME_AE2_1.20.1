# Кварцевые инструменты

<Row>

  <ItemImage id="nether_quartz_cutting_knife" scale="4" />

Режущие ножи используются для создания <ItemLink id="name_press" /> и <ItemLink id="cable_anchor" />.

Чтобы создать именную печать, щёлкните [ПКМ] по режущему ножу и вставьте слиток металла. Затем вам нужно будет
ввести название, которое вы хотите написать на печати, а затем просто извлечь готовую печать.

# Кварцевое стекло

<BlockImage id="quartz_glass" scale="8" />

В основном это прозрачное стекло, созданное с использованием <ItemLink id="certus_quartz_dust" />.
Используется для создания многих машин и предметов мода «Прикладная энергетика 2».

Есть вариант - вибрационное кварцевое стекло, которое излучает свет.
